The format of the yaml file is as follows:

      --- 
      race_name: ...,
      name_fmt: ...,
      namegen_class: ...,
      char: ...,
      color: (r,g,b),

      spawn_chance: ...,  # higher means more likely to spawn
      level_distance: ...,  # not yet supported

      hp:...,
      defense:...,
      power:...,

      ai_class: name.of.class,
